系統核心功能

功能概述

「照妖鏡」以「防護（施法加印）→ 偵測（破譯真印）→ 修復（照出真身）」三段式流程，滿足來源溯源、篡改定位與證據修復需求。

方法與技術

  • 強韌浮水印（END）：Encoder–Noise–Decoder 設計，考慮壓縮與失真。
  • 偵測（IID-Net）：BayarConv + 注意力，對局部被改特別敏感。
  • 修復流程：區塊化揭回、亮度／色彩校正，可選超解析強化。
