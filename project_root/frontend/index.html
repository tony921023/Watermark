運作原理

01

上傳原始影像

選擇要保護的 Cover 圖片，系統支援 PNG / JPG，自動調整至模型輸入尺寸。

02

強韌浮水印嵌入

Encoder-Noise-Decoder 架構將 Secret 資訊以殘差方式嵌入，PSNR > 40 dB，視覺上幾乎無差異。

03

區塊鏈指紋登記

Container 圖片的 SHA-256 雜湊值自動寫入本地區塊鏈，與前一區塊串鏈，不可竄改。

04

驗證並還原

上傳圖片時先查鏈驗證 SHA-256，通過後以 Reveal Network 解碼，輸出高品質修復圖。

區塊鏈整合

每一張圖片都有不可竄改的數位身份

嵌入浮水印的同時，系統自動計算 Container 圖片的 SHA-256 雜湊，寫入本地區塊鏈並串接前一區塊的哈希，形成完整鏈條。還原時對上傳圖片做同樣計算，若未登記或鏈完整性校驗失敗，系統直接拒絕—無法偽造、無法跳過。

  • SHA-256 圖像指紋，精確至位元組
  • 鏈式雜湊，任意竄改立即被偵測
  • 驗證前置，通過才顯示還原結果
  • JSON sidecar 儲存，不影響圖片像素

系統核心功能

強韌浮水印

END（Encoder-Noise-Decoder）設計，訓練時模擬壓縮、模糊、隨機雜訊等攻擊，確保浮水印在各種影像處理後仍可還原。

格數自選嵌入

支援 2×2、3×3、4×4 拼貼格式，格數越多韌性越強，系統還原時自動偵測最佳格數，無需手動調整。

取證報告輸出

每次嵌入與還原自動產生 HTML 報告、SHA-256 校驗清單、稽核 JSONL 紀錄，並支援 ZIP 打包下載，適合法庭舉證。
